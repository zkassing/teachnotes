# HTML标签分类

> HTML可以分为`块元素`和`行内元素`

## 块元素

块元素是一个块状结构，块元素的宽度默认就占页面的100%宽度，之后的标签会另起一行排列

*div p ul li这些都是块元素*

```html
<div>第一行</div><div>第二行</div><div>第三行</div>
```

***哪怕我们把div放在一行，这三个也是三行展示的***

## 行内元素

行内元素会紧随其后展示，不会像块元素一样换行展示

```html
<span>文字1</span>
<span>文字2</span>
<span>文字3</span>
```

***虽然是换了三行，但是页面上也只会展示一样，但是对应的回车会被替代为空格***
